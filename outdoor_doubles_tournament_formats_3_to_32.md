# Outdoor Doubles Tournament Pool Formats (3–32 Teams)

## One hard conflict
**7 teams has no clean format** if:
- 5-team pools must use 2 courts
- there can never be a lone 3-team pool
- pool sizes are only 3, 4, or 5

Everything else from **3–32** works.

## Assumption
This table is optimized for **shortest pool phase first**, then **fewest courts** when time is tied.

### Legend
- **3+3** = two 3-team pools on two courts, then crossover in wave 5
- **5** = one 5-team pool on two courts
- **4** = one 4-team pool on one court

## Notes
- Any format containing a **4-team pool** becomes a **6-wave** format.
- The cleanest **5-wave** formats are built from:
  - **5-team pools**
  - **paired 3-team pools with crossover**
- **7 teams** is the one number that breaks the current rule set.
